# Study Flow 
Study Flow is a modern Flutter application designed to help students stay organized and productive. Inspired by Folderly ( thanks kyah josh ), it provides a clean and intuitive interface for managing study materials, notes, and academic resources in one place.

## Purpose 
Study Flow aims to make studying more organized by giving students a simple and efficient way to manage their learning resources and time to do task. Whether you're keeping lecture notes, assignments, or reference materials, Study Flow helps you keep everything structured and easy to access.

This project was developed as part of a university requirement at the University of Cebu. It serves as a school project that demonstrates the use of Flutter for cross-platform mobile application development while applying modern UI/UX design principles and software development practices.
